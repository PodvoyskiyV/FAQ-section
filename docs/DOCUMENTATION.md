# Документация проекта FAQ KapitalBank

## Цветовая палитра

**ВАЖНО: Используемые цвета являются основными и не должны изменяться без согласования.**

### Основные цвета проекта:

1. **#101010** - Основной черный цвет
   - Используется для: текста вопросов, текста ответов, стрелок в иконках
   - Применение: `.faq-title`, `.question-text`, `.faq-answer p`, стрелки в SVG

2. **#FFFFFF** - Белый цвет
   - Используется для: фона страницы, фона иконки при hover/active
   - Применение: `body`, `.faq-icon svg rect` (при hover)

3. **#F3F4F6** - Серый цвет (светло-серый)
   - Используется для: фона активного блока вопроса, фона иконки по умолчанию, границ между элементами
   - Применение: `.faq-item.active`, `.faq-item::after` (границы), фон иконки в SVG

4. **#f5f5f5** - Светло-серый цвет
   - Используется для: фона кнопки при hover
   - Применение: `.faq-question:hover`

### Правила использования цветов:

- **Не изменяйте** указанные выше цвета без согласования с дизайнером
- Все цвета должны соответствовать корпоративному стилю KapitalBank
- При добавлении новых элементов используйте существующую палитру

## Статус версий

### ⚠️ Десктоп версия - ЗАФИКСИРОВАНА
**Десктоп версия зафиксирована и не должна изменяться без явного указания.**

Все стили выше медиа-запросов (`@media`) относятся к десктоп версии и считаются финальными. Изменения должны вноситься только в мобильные версии (медиа-запросы).

## Технические характеристики

### Шрифты:
- Основной шрифт: **Roboto** (подключен через Google Fonts)
- Веса: 400, 500, 600, 700

### Размеры:
- Ширина секции FAQ: **динамическая** (100% ширины экрана)
- Высота кнопки вопроса (Desktop): **80px**
- Высота кнопки вопроса (Mobile): **72px**
- Размер шрифта ответа (Desktop): **18px**
- Размер шрифта ответа (Mobile): **16px**
- Радиус скругления: **16px**
- Padding секции FAQ (Desktop): **40px** сверху/снизу, **150px** слева/справа
- Padding секции FAQ (Mobile): **24px** по периметру

### Анимации:
- Плавное раскрытие/закрытие ответов
- Вращение иконки стрелки на 180° при открытии
- Плавная смена фона при hover

### Особенности мобильной версии:
- Фон кнопки вопроса возвращается к прозрачному после закрытия
- Фон иконки возвращается к серому (#F3F4F6) после закрытия вопроса
- Иконка становится белой (#FFFFFF) только при открытом состоянии вопроса
